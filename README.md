# Ghost Vibe Coding – Heavenly Calendar

This repository contains the **Heavenly Calendar** and significant dates of the Unification Church
calculated using both the solar and lunar calendars.  
It also demonstrates how **vibe coding** (AI‑assisted code creation) can be applied to Ghost CMS  
to accelerate prototyping of integrations and extensions.

## ✨ Purpose
The goal is to combine technical innovation with spiritual meaning:  
- Provide a calendar of sacred and significant dates.  
- Show how vibe coding can generate Ghost CMS features quickly.  
- Inspire developers and communities to explore creative, uplifting uses of technology.

## ⚙️ Technologies
- Ghost CMS v6.x  
- Vibe coding (AI‑generated code guided by human intent)  
- Solar & lunar calendar calculations  
- Optional integrations: Meilisearch, FlutterFlow, APIs

## 🚀 Usage
1. Clone the repository.  
2. Install Ghost locally or connect to an existing blog.  
3. Create a template and add the code.  
4. Add a JSON file to the template folder.
